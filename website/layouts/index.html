Ledger Reconciler is a open-source command-line tool that automates

 Retrieval

Automatically logs into your financial accounts and downloads your transactions using headless Chrome.

 Classification

Classifies your transactions using Bayesian machine learning (via reckon).

 Consolidation

Updates your Ledger data file with these newly classified entries.

 De-duplication

Prevents duplicate transactions by marking the last processed transaction.
